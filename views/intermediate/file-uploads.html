Even though you are a moderator, your safety is still important! To ensure the safety of the community as a whole, we do not allow the direct uploading of code files. We cannot be sure that a file is safe without downloading it, and the risk is not worth it. The safest option is to disallow file uploads entirely.

Generally, a user uploading a file does not have malicious intentions and is not trying to violate a rule - so a friendly public reminder is often sufficient. As a note, a file named "message.txt" is often an automatic file conversion done by Discord - any message over the 2000 character limit gets converted. Keep this in mind with handling these situations.
