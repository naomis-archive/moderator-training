The answer is: D

There is a bit of a judgement call here, as spam-bots can be treated with the heaviest of ban hammers. However, we do not want to overreact to an actual user making an honest mistake, so when in doubt a suspension will suffice.

Remember that self-promotion is considered to be any off-topic link, or links to social media or blog posts. In general, links to GitHub and to personal projects are acceptable.
